# CrewAI + AgentOS Integration

This example shows how to run [CrewAI](https://github.com/joaomdmoura/crewai)
crews inside the AgentOS runtime layer.

## The Idea

CrewAI provides role-based agent teams for collaborative task execution. AgentOS
provides supervision, trace recording, secrets management, and inter-crew
messaging.

## Prerequisites

```bash
pip install crewai httpx
```

## Usage

```bash
python crewai_crew.py
```
