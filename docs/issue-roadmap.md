# Issue Roadmap

Use this file to turn the roadmap into GitHub issues.

## Good First Issues

- Document the lifecycle events emitted by the kernel.
- Improve the Python SDK README section.
- Improve the TypeScript SDK README section.
- Add dashboard screenshots to `assets/` when available.
- Add a troubleshooting section for local setup.

## Help Wanted

- Design the trace checkpoint data model.
- Review the gRPC bus protocol.
- Improve the registry health model.
- Design dashboard timeline interactions.
- Propose SDK ergonomics for existing agent frameworks.

## Architecture Needed

- Agent process isolation model.
- Tool permission model.
- Trace fork storage strategy.
- Bus backpressure strategy.
- Persistent runtime state format.

## Security Needed

- Vault threat model.
- Secret access audit events.
- Permission denial event format.
- Runtime sandboxing proposal.

## Discussion Needed

- Should AgentOS support containerized agents?
- Should the dashboard be local-only first?
- How strict should the default permission model be?
- What is the smallest useful replay checkpoint?
